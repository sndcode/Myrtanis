Myrtanis 
Version : 0.0.1 alpha C++ Recode
Author : Maurice Theißen

Info (Generally) : 
Dieses Spiel habe ich am 01.06.2015 angefangen zu programmieren.
Zuerst habe ich es bis zur Version 0.0.2 alpha in C# (.net) geschrieben,
jedoch am selben Tag das Projekt verworfen und noch einmal von vorne
in C++ angefangen.
Das alte Projekt (0.0.2 alpha .net) existiert weiterhin auf meinem Server.
Das neue Projekt wird ebenfalls zu 100% Open Source.
Der Code wird allerdings erst veröffentlicht werden, nachdem die alpha Phase abgeschlossen
wurde, um Bugs und nicht vorhandene Features auszuschliessen.

Das Spiel ist ein Text basiertes Rollenspiel, mit ein paar Karten, einigen Items zum Sammeln
und einigen Gengern zum kämpfen. NPC's sind ebenfalls angedacht.
So wird die wahl des Volkes (Ork Mensch Elf) von Beginn an eine Rolle spielen.

Es wird davon ausgegangen bzw. dazu geraten dieses Spiel mit Ruhe und bedacht zu spielen.
Ebenfalls wird eventuell eine selbst gezeichnete Karte von nöten sein, um die Übersicht zu behalten.

Map01 hat 7 Räume.
Map02 wird vorraussichtlich 10 Räume haben.
Map03 wird vorraussichtlich 15 Räume haben.

Der Projektname "Myrtanis" ist eine Abwandlung der Spielewelt des Rollenspiels "Gothic".
(Myrtana + Korinis).

Das Spiel ist eine einzelne .exe Datei , ohne Klassenbibliotheken oder Resourcen wie MP3 oder ähnlichen Files.
Alles wird entweder im Kopf vorgestellt oder mit minimalistischen Methoden dargestellt.

Die Datei , welche zum Speichern und Laden genutzt wird , ist unter diesem Pfad zu finden : 
C:\users\public\MyrtanisCppRecode\savegame.ini

Die Datei enthält Variablen und deren Werte, welche für das Spiel wichtig sind.
z.B den Namen des Spielers, die letzte Position etc.

Bitte verändern Sie diese Datei NICHT wenn Sie nicht genau wissen was Sie tun!
Es kann ein irreparabler Schaden an Ihrem Spielstand entstehen!
